VNCSniffer Source Code

Benötigt .NET, entwickelt mit Visual Studio 2022

VNCSniffer.Core: Bibliothek zur Erfassung und Analyse von VNC-Paketen
VNCSniffer.Core.Tests: Tests der VNCSniffer.Core Bibliothek
VNCSniffer.Cli: Konsolenanwendung, welche VNCSniffer.Core nutzt um VNC-Pakete in die Konsole zu schreiben (unvollständig)
VNCSniffer.GUI: Grafische Anwendung, welche VNCSniffer.Core nutzt um VNC-Datenverkehr grafisch darzustellen
